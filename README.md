# shawnsui.com

This is my personal website built with ❤️ using [Jekyll](https://jekyllrb.com), theme by [Klise](https://github.com/piharpi/jekyll-klise).

## Prerequisites

- Ruby
- Bundler
- Jekyll

## To get started

1. Install dependencies

```
bundle install
```

2.  Run the website locally

```
bundle exec jekyll serve
```
